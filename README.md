# De los datos a la toma inteligente de decisiones: ML e IA aplicadas

Material didáctico sobre Machine Learning, Data Mining e Inteligencia Artificial
aplicadas a Ingeniería Civil y de Sistemas, con enfoque en la transformación de
datos en información útil para la toma de decisiones informada.

## 🎯 Objetivo

Presentar de forma clara, estructurada y aplicada cómo los datos pueden ser
analizados y modelados mediante técnicas de ML e IA, utilizando un estudio de
caso reproducible.

El enfoque central es:

**Datos → Información → Modelo → Interpretación → Decisión**

## 🧩 Contenido

- Fundamentos de Data Mining, Machine Learning e Inteligencia Artificial
- Relación entre datos, información, conocimiento y toma de decisiones
- Flujo general de un proyecto de análisis de datos
- Técnicas principales:
  - regresión
  - clasificación
  - clustering
  - reducción de dimensionalidad
  - selección y extracción de características
- Estudio de caso reproducible en Python

## 📊 Estudio de caso

Se desarrolla un estudio de caso basado en datos de resistencia a la compresión
del concreto, abordando:

- análisis exploratorio de datos (EDA)
- preprocesamiento
- ingeniería de características
- modelos de regresión
- clasificación como extensión didáctica
- interpretación de resultados

El objetivo es ilustrar un pipeline completo de Machine Learning aplicado a un
problema real de Ingeniería Civil.

## 🚀 Uso

Este repositorio está diseñado para ser utilizado principalmente en **Google Colab**.

1. Abrir el notebook:
   - `notebooks/estudio_caso_concreto.ipynb`

2. Ejecutar las celdas secuencialmente

También es posible ejecutar el código en un entorno local utilizando:

```bash
pip install -r requirements.txt
````
